# Toolbox

This file outlines the tools and libraries used in this project.

## Core Libraries

*   **Streamlit:** For creating the web application interface.
*   **Transformers:** For loading and using the pre-trained AI model from Hugging Face.
*   **Torch:** The deep learning framework that the model is built on.

## Development Tools

*   **Git:** For version control.
*   **GitHub:** For hosting the remote repository.
*   **Visual Studio Code:** As a code editor.
